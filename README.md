# OpenTok
npm package for [OpenTok.js](https://www.tokbox.com/developer/sdks/js/)

# Installation

```
$ npm install opentok-client-sdk --save
```

# Usage

```javascript
// Include OT globally
import 'opentok';
```

Take a look at [tokbox website](https://tokbox.com/developer/sdks/js/) for documentation.
